# 🍳 AI Recipe Generator

## Overview
AI Recipe Generator is a web application that generates personalized recipes based on user-provided ingredients and preferences. The application leverages Google Gemini AI to provide intelligent recipe recommendations and reduce manual recipe search time.

## Features

- AI-powered recipe generation
- Personalized dietary recommendations
- Ingredient-based recipe suggestions
- Save favorite recipes
- Responsive user interface
- Fast performance using Vite and React

## Tech Stack

### Frontend
- React
- TypeScript
- HTML5
- CSS3
- Vite

### AI Integration
- Google Gemini AI

## Installation

Clone the repository:

```bash
git clone https://github.com/AAshi0304/AI-recipe-Generator-.git
```

Move to project directory:

```bash
cd AI-recipe-Generator-
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run dev
```
